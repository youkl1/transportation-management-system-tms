## 8. 响应式适配规则

### 8.1 断点设计
| 断点 | 屏幕宽度 | 布局 |
| :--- | :--- | :--- |
| 超小屏幕 | < 768px | 单列布局，隐藏侧边栏 |
| 小屏幕 | 768px - 992px | 双列布局，侧边栏可折叠 |
| 中等屏幕 | 992px - 1200px | 双列布局，侧边栏展开 |
| 大屏幕 | ≥ 1200px | 三列布局，侧边栏展开 |

### 8.2 适配策略
- **布局适配**：使用弹性布局和网格布局，确保在不同屏幕尺寸下的合理布局
- **组件适配**：根据屏幕尺寸调整组件大小和间距
- **内容适配**：根据屏幕尺寸调整文字大小和内容展示方式
- **导航适配**：在小屏幕下使用汉堡菜单，隐藏非关键导航项

### 8.3 移动设备适配
- **触摸友好**：按钮和可点击元素尺寸不小于48px
- **手势支持**：支持滑动、长按等手势操作
- **性能优化**：减少动画效果，优化加载速度
- **网络适配**：支持离线操作和弱网络环境
